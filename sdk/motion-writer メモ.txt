カレントディレクトリでコマンドプロンプトを開いて下記コマンドを実行

【注意】
・micro:bit v1にplenbit_eeprom_pagewriter.hexを書き込んでから行うこと

python motion-writer.py -a

-u : 書き込み状況を表示

-l : リストを列挙する
-n : モーション番号を選んで書き込む
-a : すべてのモーションを書き込む
-c : データ読み取りチェック
-m : 途中のモーション番号を選んで書き込む（最後のモーションまで）
-d : デバッグ情報を表示する
-r : バッファの読み取りチェック
-f : モーションファイルのパスを指定
